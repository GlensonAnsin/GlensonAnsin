# About Me:
A Software Developer specializing in creating both web and mobile applications using React for dynamic front-end development and Express.js for robust back-end systems. Beyond coding, I have hands-on experience with Figma, which I use to design clean, user-friendly interfaces that improve the overall user experience. I am eager to expand my knowledge, take on challenging projects, and continuously refine my skills to grow as a developer. My goal is to contribute to meaningful tech solutions while becoming the best version of myself in the IT field.
